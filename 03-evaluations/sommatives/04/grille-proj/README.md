# Grille d'évaluation — Projet d' intégration des apprentissages

## Mode d'emploi

Chaque critère est noté de 1 à 4 :

- 4 - Excellent
- 3 - Satisfaisant
- 2 - Partiellement
- 1 - Insuffisant

Total maximum : 40 points (10 critères × 4). La note finale pour le cours
sera calculée à partir de ce total et convertie sur 40%.

## Livrables 

- Expérience ludique complète intégrant images et échantillons sonores originaux dans laquelle le joueur progresse en accomplissant des actions
- Lien pour la page web du projet dans Itch.io avec documentation et un export fonctionnel.
- Dépôt Git en ligne (URL) contenant le code source et l'historique. 
  - Documentation du projet via `README.md` : arborescence, crédits, description et instructions d'opération.
