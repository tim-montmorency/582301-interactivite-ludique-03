# PROJ-EVF-6 : Validation de concept pour le projet intégrateur

Date : Avant la [séance 12](/01-deroulement/12/), avec la remise par Teams.

## Description

L'objectif de cette activité formative est de préparer un concept initial de votre projet intégrateur. Vous allez décrire le concept dans un fichier docx et je vais envoyer des rétroactions avant le cours. Voici les informations plus centrales à définir :

1. Est-ce que c'est un projet en équipe ou individuel?
2. Est-ce que vous allez continuer à développer et raffiner le jeu des projets antérieurs ? 
   1. Si oui, quels sont les changements que vous proposez de réaliser ? 
   2. Sinon, quelle est votre idée pour un nouveau jeu original ? Quels sont sa structure, ses objectifs et sa direction de présentation ? 
3. Quelle est la nouvelle fonctionnalité extra qui impacte la jouabilité (ex. high score persistant, niveau de boss, évolution du personnage, etc.) que vous voulez explorer dans ce projet ?

Envoyer le fichier docx avec cette information par le devoir Teams adéquat.
